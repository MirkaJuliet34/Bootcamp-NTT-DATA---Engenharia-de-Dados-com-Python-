# 🚀 Bootcamp de Engenharia de Dados

Bem-vindo(a) ao repositório do **Bootcamp de Engenharia de Dados**! 🎓 Este bootcamp abrange tópicos essenciais para o desenvolvimento de habilidades em programação com Python e ferramentas para visualização e análise de dados com Power BI.

## 📚 Tópicos Abordados

### 1. 🐍 Introdução ao Ambiente Python e Projetos Colaborativos
   - Configuração do ambiente de desenvolvimento: **Python**, **Jupyter Notebook**, e **VSCode**.
   - Uso de **Git e GitHub** para controle de versão e colaboração.
   - Boas práticas em projetos colaborativos de ciência de dados.

   <p align="center">
      <img src="caminho/para/sua/imagem/python-ambiente.png" alt="Ambiente Python" width="500"/>
   </p>

### 2. 📝 Sintaxe Básica com Python
   - Estrutura e sintaxe da linguagem: **tipos de dados**, **estruturas de controle** e **funções**.
   - Noções de **manipulação de strings**, **listas** e **dicionários**.
   - Introdução à **modularização de código**.

   <p align="center">
      <img src="caminho/para/sua/imagem/sintaxe-python.png" alt="Sintaxe Python" width="500"/>
   </p>

### 3. 📦 Trabalhando com Coleções em Python
   - Operações com **listas**, **tuplas**, **dicionários** e **conjuntos**.
   - Técnicas de **filtragem**, **ordenação** e **mapeamento**.
   - Uso de funções como **`map`**, **`filter`**, e **`reduce`** para manipulação de dados.

### 4. 🏛️ Programação Orientada a Objetos com Python
   - Conceitos de **POO**: classes, objetos, herança e polimorfismo.
   - Criação e uso de **métodos e atributos**.
   - Aplicação de **encapsulamento** e **abstração** em projetos.

---

## 📊 Fundamentos de Power BI

### 5. 🔍 Introdução ao Power BI
   - Configuração e principais funcionalidades do **Power BI Desktop**.
   - Conectando-se a **fontes de dados diversas**.
   - Uso de **Power Query** para transformação de dados.

   <p align="center">
      <img src="caminho/para/sua/imagem/power-bi-intro.png" alt="Power BI Intro" width="500"/>
   </p>

### 6. 📈 Visualização de Dados e Relatórios com Power BI
   - Criação de **gráficos interativos** e **dashboards**.
   - Personalização de **temas** e **layout**.
   - Aplicação de **filtros e segmentações** para refinar os dados.

### 7. 🔄 Processamento de Dados com Power BI
   - Limpeza de dados e **operações de junção** com Power Query.
   - Uso de **DAX** para criação de colunas calculadas e medidas.

### 8. 🏗️ Modelagem de Dados com Power BI
   - **Relacionamentos entre tabelas**, cardinalidade e direcionamento.
   - Modelagem de dados **estrela** e **floco de neve**.
   - Otimização para **desempenho** e implementação de **hierarquias**.
